# Spacing

#### The 5px spacing system
All outside spacing, spacing between components, is done in increments of 5 pixels. BUT this should be used as a reference and some spacings might vary, especially during the implemention. Important is to try to stay as close as possible to this numbers.

Keep in mind that spacing values sometimes donâ€™t apply to the padding within a component, like the top and bottom padding of the Buttons which is 13px.

#### Creating relationships
Elements in a design that are near each other are seen as being meaningfully related. As more space is added between elements, their perceived relationship weakens.
