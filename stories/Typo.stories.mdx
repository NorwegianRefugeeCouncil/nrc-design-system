# Typography



### Display

Use Display for the main page title.
A display is great for when you need to show an important title at the top of more structured and content-heavy pages.

<Canvas>
  <Story
    name='Display'
    parameters={{ docs: { source: { code:
      `<Text variant='display' level='1'>Display Major</Text>\n<Text variant='display' level='2'>Display Minor</Text>`
    }}}}
  >
    <VStack>
      <Text variant='display' level='1'>
        Display Major
      </Text>
      <Text variant='display' level='2'>
        Display Minor
      </Text>
    </VStack>
  </Story>
</Canvas>
