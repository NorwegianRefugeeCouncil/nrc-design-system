# Icons

### Default Icons

Based on the Activity Icons and in order to keep all icons consistent, we use the [Remix Icon set](https://remixicon.com/).
It’s an open-source neutral-style system with rounded outlines, similar to the Activity Icons. They are all clear and minimal.

The Icons components made for Designers have a Frame of 24x24px. This was done to keep the proportions when changing an icon inside a component. This is the only size we are currently using.

Default color is Black #000000 but feel free to use any color of our colors system.

### Activity Icons

NRC’s six programme activities each have their own symbol. Don’t confuse them for logos – they should never replace the official NRC logo.

More about Activity Icons at the [NRC brand book](https://identity.nrc.no/auth/?referer=%2Fbrand%23/design-guide/activity-symbols)
